# SafePrompt Privacy Policy Page

This folder contains a simple static privacy policy page for GitHub Pages.

## Files
- `index.html` — the privacy policy page

## Before publishing
Replace every instance of:
- `YOUR EMAIL HERE`

with your real contact email.

## Publish on GitHub Pages
1. Create a new GitHub repository, for example `safeprompt-privacy-policy`.
2. Upload `index.html` to the root of the repository.
3. Go to **Settings** → **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and the `/ (root)` folder.
6. Save.
7. GitHub will give you a public URL for your privacy policy page.

## Suggested URL to use in Chrome Web Store
Use the GitHub Pages URL that looks like:

`https://YOUR-USERNAME.github.io/safeprompt-privacy-policy/`
