Chrome Extension Privacy Policy

Privacy Policy for SafePrompt

SafePrompt is a Chrome extension that helps users redact sensitive text inside the ChatGPT prompt box on chatgpt.com before messages are sent.

Effective date: March 30, 2026

What SafePrompt does

SafePrompt detects and replaces sensitive text such as emails, phone numbers, names, usernames, IDs/tokens, and user-defined custom text inside the prompt composer on chatgpt.com.

What data is handled

SafePrompt may process text that appears in the ChatGPT prompt box, including text that may contain personal or sensitive information.

It also stores extension settings created by the user, such as category toggles, custom entries, replacement labels, ignored detections, minimized state, and whether protection is enabled.

How data is used

All processing is used only to provide the extension’s privacy-protection features, including detecting, replacing, and managing sensitive text in prompts.

Storage

SafePrompt uses chrome.storage.local to save user preferences and custom rules so they persist across sessions.

Data sharing

  • SafePrompt does not sell user data.
  • SafePrompt does not transfer user data to third parties.
  • SafePrompt does not use user data for advertising, profiling, or unrelated purposes.

Remote code

SafePrompt does not load remote JavaScript or remote Wasm. Its executable logic is packaged with the extension.
